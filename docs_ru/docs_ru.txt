		Создание
Для создания необходимо подключить при помощи директивы заголовок файла: #include <main.h> (необходимо прежде создать его)
Для создания и в дальнейшем использования запишем: h a = 10; h принимает только целое число(int)

		Метод del
Для удаления переменной/объекта необходимо использовать метод del(),Программа после этого метода может некоректно работать

		Метод print
Используется для вывода значения в консоль
